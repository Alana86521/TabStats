# TabStats

Ever wondered how many times you've pressed Tab while playing Minecraft? If you're a Skyblock player, the answer is probably *a lot*.

## What It Does

TabStats is a client-side Fabric mod that tracks your tab key usage:

- **Tab Presses** - Counts every time you open the player list
- **Time Open** - Tracks how long you spend looking at the tab list
- **Per-Server Stats** - Separate statistics for each server you play on
- **Achievements** - Unlock fun achievements as you hit milestones

## Commands

| Command | Description |
|---------|-------------|
| `/tabstats` | View your total and session statistics |
| `/tabstats achievements` | View all achievements and your progress |
| `/tabstats reset` | Reset your session statistics |

## Features

- **100% Client-Side** - Works on any server, including Hypixel
- **Persistent Data** - Your stats are saved locally and persist between sessions
- **Server Grouping** - Stats for `mc.hypixel.net` and `hypixel.net` are tracked together
- **Achievement System** - Creative achievement names with progress tracking
- **Clickable Chat** - Interactive buttons in chat messages

## Requirements

- Minecraft 1.21.10
- Fabric Loader
- Fabric API

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/installer/)
2. Download [Fabric API](https://modrinth.com/mod/fabric-api)
3. Drop TabStats into your `mods` folder
4. Launch the game and start pressing Tab!

## Why?

Because Skyblock players press Tab constantly for no reason, and now you can finally see just how much.
